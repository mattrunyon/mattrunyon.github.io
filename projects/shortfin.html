Title Search Tool

Shortfin is a search tool to help individuals find land titles and track who owns the properties. My role was primarily UI and UX design as well as front-end development. I ensured the usability of the system was optimal for those using it.

COLLABORATORS
  • Matthew Runyon
MY ROLE
  • Branding
  • UI Design
  • HTML
  • CSS
  • JS

IDEATION

For this particular client the timeframe was short so I jumped directly in to high-fidelity mockups in Photoshop. I explored both light and dark versions of the UI as well as various approaches to displaying relevant information.

FINAL DESIGN

The client preferred the dark UI scheme. The UI features a collapsible sidebar, and filterable table displaying results.
