MAJOR PROJECTS I'VE WORKED ON

Mechanix

Mechanix is an NSF-funded research project which utilizes sketch recognition to automatically grade free-body diagram problems in introductory statics courses. Through various methods of artificial intelligence, we are able to provide real-time feedback and guidance to students completing their homework assignments on Mechanix.

I have led project development and deployment to 5 universities with over 1000 students using Mechanix. I developed the majority of the software including the back-end API, front-end interactions, recognition algorithms, and database management.

Sketchtivity

Sketchtivity is an NSF-funded research project led by Blake Williford. It is a web-app which utilizes sketch-recognition to help teach the fundamentals of design drawing and 3D perspective drawing.

I helped improve database performance and create the tables necessary for tracking student usage and metrics. I also added new functionality including the API for a user profile to let students see their progress.

ZenSketch

ZenSketch is an award-winning educational game which takes advantage of stealth learning to improve basic line drawing quality through gameplay. It utilizes gesture-based sketch recognition to extract features related to a user's line accuracy, smoothness, and speed which plays a key role in the game.

Our team won 1st Place (Jury Award) at the CHI Play 2017 Student Game Design Competition.

Shortfin

Shortfin is a search tool developed for a client to aid in finding land titles. The project involved a large dataset which needed to undergo OCR. This was completed through parallelization in AWS. The interface also utilizes artificial intelligence to make inferences about related documents. Since the dataset is so large, lazy loading is used to make the interface snappy while providing all the data a user needs.

BitJockey

BitJockey is a software engineering class project developed for KANM, the student radio station at Texas A&M. It was designed to allow the DJs of the station to create and manage playlists from the station's digital library for use during their weekly shows. BitJockey is used by nearly 100 DJs each semester and provides the station manager with easily accessible metrics to ensure the DJs are following station guidelines.
